# 🏐 Волейбольный Симулятор 3D (Аэродинамика и Аналитика)

![Python Version](https://img.shields.io/badge/python-3.10%2B-blue)
![Streamlit](https://img.shields.io/badge/Streamlit-FF4B4B?logo=streamlit&logoColor=white)
![Plotly](https://img.shields.io/badge/Plotly-3F4F75?logo=plotly&logoColor=white)
![License](https://img.shields.io/badge/license-MIT-green)

Это не просто визуализация. Это **полноценная программная реализация математической модели полёта волейбольного мяча** в трёхмерном пространстве с учётом сложной аэродинамики. Приложение работает как интерактивный калькулятор, цифровой тренер и аналитический стенд для изучения физики ударов.

---

## 🔥 Главные возможности (Версия 5.0 - AeroSpike CV)
Проект был существенно обновлен и переписан на надежную архитектуру (Streamlit + нативный Plotly), полностью исключив проблемы с деплоем кастомных компонентов.

1. **Реалистичная физика и Аэродинамика:** Симуляция силовой (Topspin) и планирующей (Float) подачи. Учитывается лобовое сопротивление, эффект Магнуса, флаттер (эффект Кармана), а также **сила и направление ветра**.
2. **⚡ Диаграмма Энергии:** Отслеживайте кинетическую, потенциальную и полную энергию мяча в полёте, а также энергию, "съедаемую" сопротивлением воздуха.
3. **⚔️ Сравнение Траекторий:** Визуально сравнивайте две разные подачи на одном 3D-корте.
4. **🎲 Симуляция Монте-Карло и Тепловая Карта (Heatmap):** Оценка стабильности подачи с учетом микро-ошибок (вероятность успеха) и визуализация кучности падения.
5. **🛡️ Зоны Приёма:** Математический расчет того, успеет ли защитник (с учетом скорости реакции и скорости бега) добежать до мяча на основе времени полета.
6. **🔄 Обратная Задача:** Хотите подать точно в 5-ю зону? Укажите координаты, и алгоритм (Nelder-Mead) сам подберет нужную скорость, вертикальный угол и азимут удара.
7. **📄 PDF-Отчёт и CSV-Экспорт:** Выгрузка всей аналитики, графиков и сырых данных координат полета в один клик.
8. **Строгий Конструктивизм:** Дизайн в агрессивном стиле с поддержкой светлой/темной темы ОС.

## 🚀 Как запустить (Быстрый старт)

### Способ 1. Локальный запуск (Mac / Linux / Windows)
```bash
# Клонируем проект
git clone https://github.com/MOTI-jojo/AeroSpike_CV_prototype.git
cd AeroSpike_CV_prototype

# Создаем виртуальное окружение и ставим зависимости
make install
# (Или вручную: python -m venv venv && source venv/bin/activate && pip install -r requirements.txt)

# Запускаем! Приложение само откроется в браузере
make run
# (Или вручную: streamlit run app.py)
```

### Способ 2. Запуск через Docker
Если на машине установлен Docker:
```bash
make docker-run
# После сборки откройте http://localhost:8501
```

## 🌐 Деплой в облако (Streamlit Community Cloud)
Проект спроектирован так, чтобы работать в облаке **бесплатно и без сбоев**:
1. Перейдите на сайт [share.streamlit.io](https://share.streamlit.io/) и авторизуйтесь через GitHub.
2. Нажмите **"New app"**.
3. Выберите свой форк этого репозитория, ветку (`main`) и главный файл: `app.py`.
4. Нажмите **"Deploy"**. Приложение автоматически установит все зависимости из `requirements.txt` и запустится (занимает ~1-2 минуты).
