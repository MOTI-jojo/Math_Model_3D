# 🏐 Волейбольный Симулятор 3D (Аэродинамика и Аналитика)

![Python Version](https://img.shields.io/badge/python-3.10%2B-blue)
![Streamlit](https://img.shields.io/badge/Streamlit-FF4B4B?logo=streamlit&logoColor=white)
![License](https://img.shields.io/badge/license-MIT-green)

Это не просто визуализация. Это **программная реализация математической модели полёта волейбольного мяча** в трёхмерном пространстве с учётом сложной аэродинамики. Приложение работает как интерактивный калькулятор и цифровой тренер, анализируя ваши удары.

## 🔥 Главные фичи
- **Реалистичная физика:** Симуляция силовой (Topspin) и планирующей (Float) подачи. Учитывается лобовое сопротивление, эффект Магнуса и эффект Кармана (флаттер).
- **Цифровой тренер (Аналитика):** Модуль автоматически оценивает, перелетел ли мяч сетку, попал ли в площадку, насколько сложно принять такую подачу (на основе времени полета) и насколько непредсказуем планер.
- **2D и 3D Графики:** Детальный анализ падения скорости мяча и интерактивная 3D площадка с подсветкой точки максимальной скорости.
- **Строгий дизайн:** Интерфейс выполнен в агрессивном стиле конструктивизма. Нативная поддержка **Светлой и Темной** темы системы.
- **Мультиязычность:** Мгновенный перевод всего интерфейса RU/EN.

## 🚀 Как запустить (Быстрый старт)

### Способ 1. Локальный запуск (через Makefile на Mac/Linux)
Это классический путь для разработчиков:
```bash
# Клонируем проект
git clone https://github.com/your-username/Math_Model_3D.git
cd Math_Model_3D

# Создаем виртуальное окружение и ставим зависимости
make install

# Запускаем! Приложение само откроется в браузере
make run
```

### Способ 2. Запуск через Docker
Если не хотите возиться с установкой библиотек Python:
```bash
make docker-run
# После сборки откройте http://localhost:8501
```

## 🌐 Как бесплатно опубликовать (Деплой)
Чтобы этот симулятор стал доступен всем в интернете (например, для портфолио), вам не нужен сервер! Мы используем **Streamlit Community Cloud**:

1. Загрузите этот проект в свой публичный репозиторий на GitHub.
2. Перейдите на сайт [share.streamlit.io](https://share.streamlit.io/) и войдите через свой GitHub.
3. Нажмите **"New app"**.
4. Выберите свой репозиторий, ветку (`main`) и укажите главный файл: `app.py`.
5. Нажмите **"Deploy"**. Через пару минут у вас будет бесплатная публичная ссылка вида `ваш-проект.streamlit.app`!
