# Математическое моделирование подачи в волейболе (Версия 3.0: 3D Аэродинамика)

Проект представляет собой реализацию численного решения задачи Коши для математической модели подачи в волейболе с учетом сложной аэродинамики в трехмерном пространстве (3D). 

Версия 3.0 добавляет симуляцию:
1. **Эффекта Магнуса** (для силовых подач с верхним вращением — topspin), который заставляет мяч резко падать вниз.
2. **Вихревой дорожки Кармана** (для планирующих подач — float), которая заставляет мяч хаотично вилять в воздухе.

Кодовая база разработана в соответствии с современными стандартами Python, имеет строгую типизацию и разбита на модули.

## Структура проекта

```text
Math_Model_v3_3D/
├── configs/
│   ├── config_topspin.json      <-- Конфигурация для силовой подачи
│   └── config_float.json        <-- Конфигурация для планера
├── src/
│   ├── __init__.py
│   ├── physics.py               <-- Вся математика (3D кинематика, Магнус, Карман)
│   ├── visualization.py         <-- Логика 3D-визуализации (matplotlib)
│   └── input_handler.py         <-- Загрузка данных
├── main.py                      <-- Главный запускаемый скрипт
├── requirements.txt             <-- Зависимости проекта
└── README.md                    <-- Инструкция пользователя
```

## Требования
- Python 3.8 или выше.

## Инструкция по установке

### Шаг 1: Создание виртуального окружения
Откройте терминал и перейдите в папку с проектом `Math_Model_v3_3D`. Выполните команду:
```bash
python -m venv .venv
```

### Шаг 2: Активация окружения
- На **Windows**:
  ```bash
  .venv\Scripts\activate
  ```
- На **Linux / macOS**:
  ```bash
  source .venv/bin/activate
  ```

### Шаг 3: Установка библиотек
```bash
pip install -r requirements.txt
```

---

## Как запускать программу

### Режим 1: Чтение из конфигурационного файла (Рекомендуемый)
Для запуска силовой подачи:
```bash
python main.py --config configs/config_topspin.json
```
Для запуска планирующей подачи:
```bash
python main.py --config configs/config_float.json
```

### Режим 2: Интерактивный ввод
Если вы хотите ввести параметры через консоль:
```bash
python main.py
```

## Результаты
Программа выведет в консоль расчетную начальную скорость и силу удара в Ньютонах, а затем построит **трехмерный график (3D)** полета мяча над площадкой и сохранит его в директорию `graphics/`.
